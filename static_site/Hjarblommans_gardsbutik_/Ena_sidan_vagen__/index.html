Ena sidan vägen.

Från parkeringen följ skogslinjen bort från huset ner mot bäcken. Du kommer först till ett block som man ser från vägen. Följ den fina väggen för att komma till mer klättring. När väggen tar slut går det att fortsätta över åkern till nästa sektor.
