Ringhals

Fint område vid havet. Ett stenkast från Ringhals kärnkraftverk. Parkera på den lilla parkeringen vid skylten. Gå ner mot vattnet och vik av vänster. Du ser klipporna. Här var det tydligen bouldrande surfare som började känna på greppen för ca 10+ år sedan (var har vi hört den förut?). Happybouldergänget har nyligen gjort lite problem i blockhavet. Annorlunda klippa med annorlunda grepp. Finns endel block med branta tak, väggar, areten, dynos, lätta grepprika problem etc. Finns något för alla. Tänk på att det är ett naturreservat. Var rädd om naturen. Borsta minimalt. Låt träd och buskar stå orörda. Släng inget skräp. Plocka inga blommor. Ingen chipping. Stör INTE fågelskådarna...vi kan trivas ihop! Bada INTE i vattnet utanför klipporna.
