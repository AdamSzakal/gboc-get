‹ Milbanan 57.6883, 12.0576

Beskrivning

Ett par fina block mellan milbanan och 8an. utmed stora delsjön. Parkera på Brudarebacken parkering och gå rätt exakt 1km för att komma fram. Det stora blocket ligger bara några meter från stigen som går längst med vattnet. Kollar man upp till vänster borde man se det om det inte är allt för mycket löv ivägen.
