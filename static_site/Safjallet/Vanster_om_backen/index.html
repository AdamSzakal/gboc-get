Vänster om backen

Liten vägg som ligger till vänster om backen som går upp till vattenreservoaren (eller vad det nu är). Ligger rakt ovanför grusvägen på bilden med husen, dock bättre att gå runt via backen för att inte störa. Potential för ett par problem, lite taskig landning dock.
