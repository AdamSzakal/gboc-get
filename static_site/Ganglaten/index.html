Gånglåten

Litet område med en del intressanta problem värda att nämnas. Några lätta, några fina, några projekt, en perfekt mix med andra ord! Det ligger precis vid området gånglåten nära Frölunda torg. Från Frölunda torg ta er norrut förbi kulturhuset på höger hand och sedan skolan på vänster hand via cykelvägen. Huvudklippan (Stora väggen) dyker upp på höger hand efter ca. 300 meter efter skolan.
