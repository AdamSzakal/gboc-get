El perro punteado (5+)

Börja på den braiga listen/juggen fem meter till höger om Baba. Rätt upp via sloperkant. Sprickan till höger ingick inte vid FA. Vänstervariant finns. Börja isåfall stående till vänster om själva areten och jobba rakt upp på flaket. Något lättare än högervarianten.
