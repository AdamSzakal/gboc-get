‹ South beach undefined

Beskrivning

Vattenslipad brant klippa med stora sköna grepp, landning på sandstrand och kvällssol. Vad mer behövs? Följ huvudvägen via vänstersväng genom Fiskebäckskil och vidare på Kristinebergsvägen till parkeringsplatsen vid ladan. Till vänster om ladan skymtar ett litet gult hus som man till fots passerar på dess högra sida. Följ ravinen till stranden. (Det kan vara folk i huset och då är det bättre att ta vägen över höjden till höger om ravinen istället för att passera precis vid tomtgränsen.)
