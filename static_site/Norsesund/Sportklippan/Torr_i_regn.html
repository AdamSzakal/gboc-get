‹ Torr i regn

Beskrivning

Ca 3.5m högt stenblock, svagt överhängande, med varierade grepp på dess nordvästra sida. Kan vara rolig uppvärmning för närliggande sportleder. Ligger 5m från vägen precis vid stigen in mot stora klippan. Start på vass jug till vänster av bouldern, traversera högerut till halvkrimpar, gå via sloper till höger upp rakt upp mot kanten av blocket. Slutmatcha kanten(5) eller mantla upp(6?) om du törs.

Grad: 5+

Betyg: ★
