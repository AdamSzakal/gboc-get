‹ Var är hålet?

Beskrivning

Start längst till vänster på "stupväggen" Den lilla väggen vid klapperstens fältet. HH på liten pocket, VH på list. Fötterna på väggen. Sen rakt upp via ett mini hål. Används flakkanten för H fot blir den lättare.

Grad: 5+
