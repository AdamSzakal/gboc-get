Roys arete (6A)

Överhängande sloperarete från höger till vänster. Ligger 1m till höger om Frustrerande vid värme. Svingstart eller ej. Bestäm själv. Video: https://instagram.com/p/8X6HH3kx9F/ Bra för ensambouldraren. Parkera vid ICA. Gå över vägen på övergångsstället mot Poseidon. Sväng vänster mot de röda husen in på cykelstigen. Efter ca 50m sväng upp åt höger mot toppen på kullen. Problemet ligger längst upp på samma sten som Frustrerande vid värme. Problemet är gjort sedan tidigare och kallas garanterat något annat. Säg till bara så ändrar vi. Jag valde namnet efter den starka Roy Andersson-känslan det rödbruna radhusområdet ger mig.
